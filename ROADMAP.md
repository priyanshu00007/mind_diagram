# Roadmap

## Near-term (weeks)
- Diagram rename & delete from sidebar context menu
- Drag-and-drop reorder diagrams in sidebar
- Undo/redo stack for AI edits
- Theme persistence per diagram
- Export as Markdown with embedded Mermaid
- Dark/light/system theme toggle
- Keyboard shortcuts (Cmd+S save, Cmd+Z undo, etc.)
- Auto-save indicator improvements
- Resizable editor/preview split
- Collapsible AI chat panel

## Medium-term
- Import diagrams from Mermaid code pasting
- Export to PNG/SVG/PDF with custom dimensions
- Diagram templates (blank, microservice, flowchart, sequence)
- Cloud sync via backend API
- Collaborative editing with live cursors
- Version history comparison (visual diff)
- Shareable diagram links

## Long-term
- AI auto-layout (rearrange messy diagrams)
- Natural language editing ("add Redis cache here")
- Import from GitHub repo URL
- Embeddable diagrams for Notion/README/Confluence
- Presentation mode with step reveals
- Architecture scoring (cost, latency, fault tolerance)
- Custom themes and theme marketplace
- Terraform/K8s manifest visualization
- VS Code extension
- CLI tool for CI/CD pipeline
- Interactive simulation (traffic flow, failure scenarios)
- Offline PWA support
- Mobile companion app
